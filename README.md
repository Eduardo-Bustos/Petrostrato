# PetroStratum

Advanced macro-financial and energy system simulation platform.

PetroStratum models systemic pressure, synchronization gaps, and cross-asset transmission under geopolitical stress scenarios.

---

## 🚀 Core Capabilities

- Real-time market data ingestion
- Feature engineering (R*, Θ, SG)
- Scenario simulation
- Monte Carlo forecasting
- Streamlit dashboard

---
