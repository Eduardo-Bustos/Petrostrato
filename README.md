# PetroStratum

Institutional-grade macro-financial and energy system simulation platform.

PetroStratum is designed to model systemic pressure, synchronization gaps, and cross-asset transmission under geopolitical and macroeconomic stress scenarios. The system integrates real-time data ingestion, econometric modeling, and Monte Carlo simulation to deliver high-fidelity decision intelligence.

---

## 🚀 Core Capabilities

- Real-time market data ingestion (energy, metals, macro indicators)
- Feature engineering aligned with systemic variables (R*, Θ, SG)
- Scenario simulation (geopolitical, monetary, supply shocks)
- Monte Carlo multi-path forecasting
- Cross-asset correlation and transmission analysis
- Interactive Streamlit dashboard (institutional layout)
